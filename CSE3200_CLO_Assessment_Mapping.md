## Course Contents
Students will work in groups or individually to develop a term project to solve complex engineering problems. It may include I/O drivers, operating systems modules, or software solutions of practical problems involving complex engineering processes. Students will develop a software system using recent technologies/programming while maintaining software engineering principles with proper documentation.

## Learning Objectives
By the end of this course/presentation, students should be able to:

1. Analyze complex engineering problems and identify system requirements.
2. Design and develop a complete software/system solution using appropriate methodologies.
3. Apply modern tools and technologies in system implementation.
4. Test, validate, and evaluate system performance and reliability.
5. Work effectively in teams and manage project tasks professionally.
6. Prepare technical documentation and deliver effective presentations.

---

## Course Learning Outcomes (CLOs)
After completing this course, students will be able to:

1. **CLO1: Problem Analysis**  
   Identify, formulate, and analyze complex engineering problems to define appropriate system requirements.
2. **CLO2: System Design & Development**  
   Design and develop a complete software/system solution (e.g., OS module, I/O driver, or real-world application) using appropriate methodologies and modern technologies.
3. **CLO3: Application of Modern Tools**  
   Select and apply suitable modern tools, programming languages, frameworks, and development environments for system implementation.
4. **CLO4: Investigation & Testing**  
   Conduct testing, validation, and performance evaluation of the developed system to ensure correctness, reliability, and efficiency.
5. **CLO5: Teamwork & Project Management**  
   Work effectively as an individual or in a team to plan, manage, and execute a system development project following software engineering principles.
6. **CLO6: Communication & Documentation**  
   Prepare comprehensive technical documentation and present project outcomes effectively to stakeholders.

## 3) Detailed Rubrics for Each CLO (BAETE)

### Performance Levels

| Level | Description | Score Range |
|---|---|---|
| Excellent (4) | Exceeds expectations with strong depth and quality | 80-100% |
| Good (3) | Meets expectations with minor limitations | 65-79% |
| Satisfactory (2) | Basic achievement, lacks depth | 50-64% |
| Poor (1) | Fails to meet minimum requirements | <50% |

### CLO1: Problem Analysis (10 Marks)

| Criteria | 4 (Excellent) | 3 (Good) | 2 (Satisfactory) | 1 (Poor) |
|---|---|---|---|---|
| Problem Definition | Clear, precise, well-scoped | Mostly clear | Vague or partially defined | Unclear |
| Requirement Analysis | Complete & justified | Mostly complete | Some missing | Incomplete |
| Feasibility Study | Comprehensive analysis | Moderate | Limited | Not done |

### CLO2: System Design & Development (20 Marks)

| Criteria | 4 (Excellent) | 3 (Good) | 2 (Satisfactory) | 1 (Poor) |

| Design Tools (UML, etc.) | Complete & professional | Mostly complete | Partial | Missing |

## Example Scenario
1. **Project Title:** Smart Traffic Monitoring System
2. **Group:** G1 (3 students)
3. **Student Evaluated:** Student A
4. **Total Marks:** 100

### 1) Assessment Scores (Raw Marks Given by Instructor)

| Assessment | Max Marks | Obtained Marks |
|---|---:|---:|
| A1: Proposal & Problem Analysis | 10 | 8 |
| A2: System Design | 15 | 12 |
| A3: Implementation | 20 | 16 |
| A4: Testing | 15 | 11 |
| A5: Documentation | 15 | 13 |
| A6: Presentation & Viva | 15 | 12 |
| A7: Teamwork & Management | 10 | 9 |
| **Total** | **100** | **81** |

### 2) CLO-wise Attainment Calculation

- **CLO1 (Problem Analysis) -> A1**  
  Score = 8/10 = 80% -> **Level 4 (Excellent)**

- **CLO2 (Design & Development) -> A2 + A3**  
  A2 contribution = 12/15  
  A3 contribution (design part) = 8/10 (assumed split)  
  Total = (12 + 8) / 20 = 20/20 = 100% -> **Level 4**

- **CLO3 (Modern Tools) -> A3**  
  Tool-related marks = 8/10  
  Score = 8/10 = 80% -> **Level 4**

- **CLO4 (Testing & Investigation) -> A4**  
  Score = 11/15 = 73% -> **Level 3 (Good)**

- **CLO5 (Teamwork & Management) -> A2 + A7**  
  A2 teamwork portion = 4/5  
  A7 = 9/10  
  Total = (4 + 9) / 15 = 13/15 = 86% -> **Level 4**

- **CLO6 (Communication) -> A5 + A6**  
  A5 = 13/15  
  A6 = 12/15  
  Total = 25/30 = 83% -> **Level 4**

### 3) Summary of CLO Attainment

| CLO | Marks | Percentage | Attainment Level |
|---|---|---:|---|
| CLO1 | 8/10 | 80% | 4 (Excellent) |
| CLO2 | 20/20 | 100% | 4 (Excellent) |
| CLO3 | 8/10 | 80% | 4 (Excellent) |
| CLO4 | 11/15 | 73% | 3 (Good) |
| CLO5 | 13/15 | 86% | 4 (Excellent) |
| CLO6 | 25/30 | 83% | 4 (Excellent) |

### 4) Interpretation (BAETE)
- All CLOs are attained (>=60%).
- Strong performance in:
  - Design & Development (CLO2)
  - Communication (CLO6)
- Slight improvement needed in:
  - Testing & Validation (CLO4)

### 5) Rubric-Based Justification Example
**Example for CLO4 (Testing):**
- Test cases were present but not exhaustive -> Level 3.
- Performance analysis was moderate -> Level 3.
- Hence, overall rubric level = **Good (3)**.
